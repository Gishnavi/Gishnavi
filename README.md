# Gishnavi — profile & portfolio

Personal portfolio site: modern, responsive, and easy to deploy (e.g. [GitHub Pages](https://pages.github.com/)).

## Quick start

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize

- Edit copy and links in `index.html`.
- Adjust colors and motion in `css/styles.css`.
- Tweak interactions in `js/main.js`.
